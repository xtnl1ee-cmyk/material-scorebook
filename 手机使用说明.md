# 手机免费使用说明

这个网站是纯静态网页，不需要服务器和数据库，适合一个人免费使用。

## 最推荐：GitHub Pages

1. 注册或登录 GitHub。
2. 新建一个仓库，例如 `material-scorebook`。
3. 上传这个文件夹里的 `index.html`、`style.css`、`app.js`、`data.js` 和 `assets` 文件夹。
4. 进入仓库 `Settings` -> `Pages`。
5. `Source` 选择 `Deploy from a branch`，分支选 `main`，目录选 `/root`。
6. 保存后等待一两分钟，GitHub 会给你一个网址。
7. 用手机打开这个网址，再添加到主屏幕即可。

## 重要提醒

- 评分、观后感、时长、物料修改都保存在当前浏览器本地。
- 换手机、清浏览器缓存、换浏览器前，先点网站右上角“导出”备份。
- 在新手机上打开网站后，点“导入”恢复备份。

## 不想上传到网上

也可以把整个文件夹发到手机上，直接打开 `index.html`。但有些手机浏览器会限制本地文件读取图片，所以长期使用更推荐 GitHub Pages。
